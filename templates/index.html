detikcom Search API Documentation

Search Endpoint

GET /search

Parameters

q (required)

The search query to look for on detik.com

pages (optional, default: 3)

Number of pages to scrape (min: 1, max: 10)

Example Usage

Basic search:

GET /search?q=teknologi

Search with custom page count:

GET /search?q=teknologi&pages=2

Response Format

Success Response (200 OK):

{
    "status": "success",
    "query": "teknologi",
    "total_results": 20,
    "results": [
        {
            "title": "Article Title",
            "image_url": "https://example.com/image.jpg",
            "body_text": "Article preview text...",
            "publication_time": "2024-03-10T12:00:00",
        },
        ...
    ]
}

Error Response (400/500):

{
    "status": "error",
    "message": "Error description"
}
